# AI Journal with Productivity Judgement

This project is a personal AI-powered journal that stores journal entries with embeddings and uses an OpenAI model to rate daily productivity with advice.

## Features

- Save journal entries with semantic embeddings
- Retrieve similar past entries
- AI rates your productivity and provides advice

## Requirements

- Python 3.9+
- OpenAI Python client (`openai` package)
- Set your `OPENAI_API_KEY` as an environment variable
